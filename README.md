## 📱 Projects

### 🏡 하루마을 (HaruVillage)
> 하루를 기록하고, 나만의 마을을 가꾸는 생활 습관 형성 안드로이드 앱

<img src="https://img.shields.io/badge/Google%20Play-정식%20출시-3DDC84?style=flat-square&logo=googleplay&logoColor=white"/>
<img src="https://img.shields.io/badge/다운로드-1%2C400%2B-blue?style=flat-square"/>
<img src="https://img.shields.io/badge/평점-4.9%2B%20⭐-yellow?style=flat-square"/>
<img src="https://img.shields.io/badge/운영-1년%209개월-orange?style=flat-square"/>

**주요 기능**
- 📓 감정 선택·사진 첨부 스마트 다이어리 (자동 저장, 감정 필터, 사진보관함)
- 🏘️ 맵 34종·펫 36종·아이템 112종으로 나만의 마을 꾸미기
- 🤖 OpenAI GPT-3.5-turbo 기반 AI 촌장 인사
- 🚶 Samsung Health Connect 연동 걸음 수 추적 및 보상
- 👥 전체채팅·1:1 개인채팅·이웃마을 방문 커뮤니티
- 🎮 숫자게임·스도쿠·컬링 3종 미니게임 + 글로벌 랭킹
- 📚 사자성어·영어단어·상식퀴즈 학습 콘텐츠
- 🔔 AlarmManager 맞춤 알림, ForegroundService 상시 걸음 추적

**기술적 도전**
- Flow 기반 2단계 저장 구조 (Room 즉시 저장 → Firestore 동기화)로 데이터 유실 방지
- ForegroundService + BootReceiver로 배터리 최적화 환경에서도 걸음 수 상시 추적
- Room Migration(v1→v4) 스크립트로 앱 업데이트 시 사용자 데이터 무결성 보장
- Firebase Firestore 한계 극복을 위한 Spring Boot + MySQL API 서버 마이그레이션 진행 중
- Firebase Auth `linkWithCredential`로 게스트 → Google 계정 무중단 전환 구현

### 👴 어르신을 부탁해
> 특허·논문 기반의 노인 안심 케어 및 실시간 모니터링 안드로이드 시스템

<img src="https://img.shields.io/badge/동국대-캡스톤디자인%20총장상(1등)-gold?style=flat-square"/>
<img src="https://img.shields.io/badge/특허-출원-blue?style=flat-square"/>
<img src="https://img.shields.io/badge/논문-학술대회%20발표-green?style=flat-square"/>

**주요 기능**
- 📍 Kakao Maps + GPS 기반 보호자↔피보호자 실시간 위치 모니터링
- 🔔 FCM을 활용한 일반 알림 / 위험 상황 경고 알림 이원화
- 🦶 STEP_COUNTER 센서 기반 활동량 측정 및 이상 감지
- 📡 Haversine 공식으로 GPS 좌표 간 거리 계산 및 안심 구역 이탈 감지
- 📋 보호자 앱 ↔ 피보호자 앱 간 1:1 매핑 관리 구조

**기술적 도전**
- STEP_COUNTER 센서 데이터의 기기 재부팅 시 초기화 문제를 SharedPreferences 보정 알고리즘으로 해결
- 위치 기반 출석 자동화: Haversine 공식 적용으로 오차 범위 내 자동 출석 처리
